Inventory Management System

Having a system to help you in the management of inventory(stock),products and customers is one in a life time advantage from many business.The Inventory Management System, can do such that.It helps you keep a record of products that are available in stock, the number of customers and the number of orders per customer.Having this system will reduce your work load significantly.


The Inventory Management System can be used by any one, it has multiple pages that are easy to navigate to with the help of the main page.The main page contains a link to each page that the system contains.These is one of the reasons why this system is so easy to use. You can just go from one page to another by a click of a button, and just like that your navigating through the pages. The Inventory Management System is user-friendly, and it can be used by anyone.


The Inventory Management System was developed using languages such as Java, and SQL using MySQL as the DBMS(Database Management System) relational database system.The serve used is the Apache serve.With the help of one of the most used softwares by the name of xampp. Xampp is a cross platform software that helps with the management of relational databases.


These system provides security, meaning that it can not be used by anyone.Every person that uses this system has to have a username and a password, and not have one means that you will not have access to any information that is stored using this system.On top of that it provides form validation to prevent a person from logging in using unknown credentials.



The main purpose of this page is to help the user navigate easily throughout the application, because this page has a link to each page in the application,this makes it easy to navigate as the user only has to click the link to navigate to the next page.

The product page helps to manage all the products that are available. This page has a product table, that shows a list of products in order, and a new product can be added using the from next to the table.

The customer page helps to manage all the customers. This page has a customer table, that shows a list of customers in order, and a new customer can be added using the from next to the table.

The category page helps to manage all the product category's. This page has a category table, that shows a list of category's in order, and a new category can be added using the from next to the table.

The user's page helps to manage all the user's of the system. If a person has a password he/she is able to login, otherwise a password has to be made. A new user can be added using the form next to the table.

The order page helps to manage all the orders. This page has customer and product tables with data taken from the original tables. Using this page you can add an order form a particular customer, and that order is then taken to the order table.

Electronics Management System

Electronics management system, is an important system/application to have especially when your working with many customers.The system reduces paper work as it stores everything on the database,because it stores data on the database it is not likely to lose such data. That is not all,this system also produces a printable slip/document that you can print and give to a customer.The slip contains all the necessary information about the device. This system can help a lot people that are in the repair business.

If there is data in the database, the table next to the form will retrieve it and display it. because of this the system becomes more user friendly.

When the customer pays the amount due the system produces a printable slip,and the new data is recorded in the database showing the new due amount as zero.The zero shows that the customer has fully settled he/her debt.

Every time a customer is recorded his/her data is displayed in the records table. The process is the same for each customer that is to be recorded in the system.
